## 🐍 AI Snake Game

This is an AI-based Snake Game developed using Python and Pygame.

## 🎮 Features
- Automatic snake movement using AI agent
- Food chasing logic
- Collision detection (wall & self)
- Score tracking
- Restart option
- Increasing difficulty

## 🤖 AI Logic
The agent decides the movement based on the position of the snake and food. It also avoids unsafe moves like collisions.

## 💻 Technologies Used
- Python
- Pygame

## 🚀 How to Run
1. Install pygame:
   pip install pygame
2. Run the game:
   python snake_game.py
